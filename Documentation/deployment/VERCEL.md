# Vercel Deployment Guide (Non-Technical)

Vercel is the easiest option for beginners.

1. Go to https://vercel.com
2. Sign in using your GitHub account
3. Click **New Project**
4. Select your forked repository
5. Fill the environment variables section
6. Click **Deploy**

Vercel will:

- Install dependencies
- Build the app
- Publish it online

You will receive a public URL in minutes.

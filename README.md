# Mustaheq Frontend App

## Start Development

- git clone repo
- cd directory
- `yarn` or `npm i`
- clone .env.example => .env
- Set backend url to either local `http://127.0.0.1:8000` or to deployed `https://api.mustaheq.org/`
- Set storage directory either to dev one `https://pdt-bucket.s3.us-east-1.amazonaws.com` or to production of `https://api.mustaheq.org/`
- `yarn start`
